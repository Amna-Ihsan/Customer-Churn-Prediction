## Project Overview

This project is a **Customer Churn Prediction App** that uses machine learning to predict whether a bank customer is likely to leave or stay. The model is trained using customer information such as credit score, age, gender, geography, balance, tenure, number of products, credit card status, active membership, and estimated salary. A Random Forest model is used for prediction, and the final model is connected with a Streamlit web app where users can enter customer details and get churn predictions easily.
